    Calendar

    Extends:

    react~Component → Calendar

    Calendar is responsible to list the available years, months, and handle the user's choice, when one of them is selected.

      Children:
    • YearSelector
    • MonthSelector
    • PrintSheet

    React Props:

    NameTypeAttributeDescription
    fetchAllData function this will initiate the ALL_DATA_REQUESTED redux action
    yearSelected function this will initiate the YEAR_SELECT_REQUESTED redux action
    monthSelected function this will initiate the MONTH_SELECT_REQUESTED redux action
    selectedYear number loads through redux state. Initially it will be the actual year but it can be modified by clicking on the corresponding year
    selectedMonth number loads through redux state. Initially it will be the actual month but it can be modified by clicking on the corresponding month
    yearList number[] static array which contains the year before the actual year, the actual year, and the year after it
    monthList string[] static array which contains the actual months of the local year
